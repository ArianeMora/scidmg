.. _installing:

Installing sci-dmg
==================

Note: It is strongly recommended to create a `virtual environment <https://packaging.python.org/guides/installing-using-pip-and-virtual-environments/>`_
for each version of sci-dmg installed.

.. code-block:: python

    pip install scidmg


.. code-block:: R

    install.packages("scidmg")


Below are supported versions of sci-dmg:

.. list-table::
   :widths: 10 10
   :header-rows: 1

   * - sci-epi2gene Release
     - Version
   * - 1
     - 1.0.0
